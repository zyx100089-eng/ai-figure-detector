# Real or Rendered? Detecting AI-Generated Scientific Figures

Can we tell the difference between real scientific figures from published papers and figures generated by AI? This project investigates detection methods across two categories of AI-generated scientific figures:

- **Pixel-level fakes**: Figures generated directly by image models (DALL-E, Stable Diffusion)
- **Code-level fakes**: Figures produced by executing LLM-generated plotting code (Claude, GPT-4, Llama)

We compare three detection approaches — fine-tuned CNNs, frequency domain analysis, and statistical consistency checks — and evaluate which generation methods are hardest to detect.
